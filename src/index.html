fastberry/apachephp Container

Congratulations! If you see this message the apache server is successfully running in the fastberry/appachephp container.

In order to see your own web content do this:

Stop the docker container - go to a terminal window and type: 

       docker -kill apachephp
Create a directory and put your files there. Then run the docker container again using these commands, giving the name of your directory instead of "/home/pi/yourdirectory": 
     docker rm apachephp
     docker run --name apachephp -it -p 80:80 -v /home/pi/yourdirectory:/var/www/html fastberry/apachephp
You can update your web content by editing your files in the web content directory, or copying files from another location.
